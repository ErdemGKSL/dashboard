# Development

## Prerequisites

- Install nodejs

- Install pnpm if not already installed:

  ```bash
  npm i -g pnpm
  ```

## Development Server

1. Install dependencies:

   ```bash
   pnpm i
   ```

2. Start the dev server:

   ```bash
   pnpm dev
   ```

If you need the dashboard to connect to a running ferrumc instance specify the WebSocket port via the URL query string, for example: `?ws_port=9000`

## Building

### Building Notes

- You do not need to produce a single bundled application for release; Workflow handles the release.
- Ferrumc can fetch the release zip generated by the build workflow. If you push a new build just recompile the main ferrumc binary, the dashboard will update automatically.

### How to build

1. Install dependencies (if not already done):

   ```bash
   pnpm i
   ```

2. Build the project:

   ```bash
   pnpm build
   ```

The build output will be placed in the `./build` directory and should resemble the following structure:
